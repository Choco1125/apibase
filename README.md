# apibase

Template to restful api projects

## Installation

```bash
npm install
```

```bash
cp .env.example .env
```

## Run app

### Docker 🐳

_It takes the configuration from the .env file_

```bash
npm run docker:run
```

### Migrations 🗳️

```bash
npm run migrations:run
```

### App

_With nodemon_

```bash
npm run dev
```

## Utils

### Generate migrations

```bash
npm run migrations:generate foo
```
